Autoblock spam users

In order to prevent illicit content (spam messages) from being published on the platform, a mechanism will be implemented to auto-block users. When a user is blocked, their content becomes inaccessible (if they are unblocked, it is accessible again).

  • Allow or disable user auto-blocking.
  • Allow defining the weight (in integers) of the rules.
  • Allow defining the threshold from which the autolock is applied (>=2 for example).
  • Add links to be able to list users who meet these rules (optional).
  • Define whitelist rules that would bypass autoblocking.
